# Docker Compose Task

## Objective
Understand how to run multi-container applications using Docker Compose.

## What I Did
- Created a docker-compose.yml file
- Defined multiple services (web and app)
- Built and ran containers using docker compose
- Verified running services
- Stopped and cleaned the environment

## Services
- Web: Nginx container exposed on port 8080
- App: Apache HTTP server container

## Commands Used
- docker compose up -d
- docker compose ps
- docker compose down

## Proof
Screenshots show the running services.
